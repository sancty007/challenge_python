## Instructions pour les contributeurs :

1. **Créez une branche pour votre solution :** Avant de commencer à travailler sur votre solution, créez une nouvelle branche à partir de la branche principale (main) pour isoler vos modifications.

    ```bash
    git checkout -b nom-de-votre-branche
    ```

2. **Développez localement :** Clonez ce dépôt localement sur votre machine et travaillez sur votre propre copie du projet. Assurez-vous de récupérer les dernières modifications avant de commencer à travailler sur votre solution.

    ```bash
    git clone https://github.com/votre-utilisateur/challenges.git
    ```

3. **Soumettez une pull request :** Une fois que vous avez terminé votre solution, soumettez une pull request pour proposer vos modifications à l'équipe. Décrivez vos changements et assurez-vous de passer en revue les modifications avant de les soumettre.

4. **Communication :** N'hésitez pas à poser des questions, à partager des idées et à discuter des solutions proposées avec les autres membres de l'équipe. La communication ouverte est essentielle pour garantir la qualité et la cohérence des solutions.

5. **Licence :** En proposant une solution, vous acceptez que votre code soit sous licence MIT et puisse être utilisé librement par d'autres.

Bonne résolution de challenge !
